## 📋 用户需要操作

### 方案 A：Vercel 部署（推荐用于测试站）

1. **登录 Vercel**
   - 访问 https://vercel.com
   - 使用 GitHub 账号登录

2. **创建项目**
   - 点击 "Add New..." → "Project"
   - 选择仓库 `MerakW/merakt.cn`
   - Framework Preset: **Astro**（自动识别）
   - 点击 "Deploy"

3. **等待部署完成**
   - 首次部署约 1-2 分钟
   - 部署 URL：`https://merakt-cn.vercel.app`

4. **配置自定义域名（可选）**
   - Project Settings → Domains
   - 添加你的域名

---

### 方案 B：GitHub Pages 部署（备选）

1. **启用 GitHub Pages**
   - 进入仓库 Settings → Pages
   - Source: 选择 "GitHub Actions"
   - 保存

2. **推送代码**
   ```bash
   cd /home/merak/.openclaw/workspace/merakt.cn
   git push origin main
   ```

3. **等待部署**
   - 进入 Actions 标签页查看进度
   - 部署 URL：`https://MerakW.github.io/merakt.cn/`
